# Our Perspective
We are competing with a partnership between moen and farmers. This partnership creates an offer to install the moen flo for $745. Moen has partnered with Frontdoor.com to provide plumbers who will install the moen flow for a discount. We looked into the program and decided against joining because the payout from frontdoor is very low for the install. It forces a plumber to nickle and dime the customer in order to make any margin. Frontdoor has a bunch of "skus" that define extra money the plumber can earn like for having to reposition a pressure reducing valve or for adding a loop to fit the moen flo on the line when there is not enough room. So, the plumber shows up with the customer expecting it to be $745 but in the end it costs much more because the plumber has to add on costs in order to make a margin and the standard install frontdoor/moen/farmers define is nearly impossible on any main line coming in. This makes the plumber look predatory. But the big point here is that the customer will almost always feel a bait and switch and that will look especially bad for the plumber but also for the farmers agent that told the homeowner about the plan.
